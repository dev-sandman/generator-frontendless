* [Apache Tomcat](www.com) >= version 8.0.0

* [Java](www.com) >= version 8.0.0

* [Bower](www.com) >= version 3.1.7

## Dependencies

* AngularJS
* D3
* Grunt
* Karma
* Jasmine
* Protractor  

## Enviroment configuration

With all requirements installed running in npm:

    npm install

after running:

    bower install

## Build & Development

For build running:

    grunt build

For preview running:   

    grunt serve

## Tests

For tests running:

    grunt tests

* Unit Test  

      grunt unit-test

* Interface Test   

      grunt e2e-test
